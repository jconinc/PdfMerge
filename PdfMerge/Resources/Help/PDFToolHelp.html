PDF Tool Help

PDF Tool is a macOS utility for working with PDF files. Select a tool from the sidebar to get started.

Tools

Merge (Cmd+1)

Combine multiple PDF files into one. Drag and drop files, then reorder as needed.

Split (Cmd+2)

Split a PDF by page ranges, every N pages, or by selecting individual pages.

Rotate (Cmd+3)

Rotate individual pages or all pages at once. Select pages and choose a rotation direction.

Compress (Cmd+4)

Reduce PDF file size. Choose a preset: Screen (smallest), eBook, Printer (default), or Prepress (highest quality).

Extract Pages (Cmd+5)

Extract specific pages from a PDF. Select pages visually or type a range like "1, 3-7, 12".

OCR (Cmd+6)

Add searchable text to scanned PDFs, or extract text content. Uses Apple's built-in text recognition.

Annotate (Cmd+7)

Add highlights, underlines, text notes, shapes, and freehand drawings to a PDF.

Fill Form (Cmd+8)

Fill in PDF form fields. Supports text fields, checkboxes, radio buttons, and dropdowns.

Convert (Cmd+9)

Convert PDFs to images or images to PDF. Advanced options include Word and Excel conversion.

Print (Cmd+0)

Preview and print a PDF using the standard macOS print dialog.

Protect / Unlock

Add password protection and permissions to a PDF, or remove a password from a protected PDF.

Keyboard Shortcuts

  • Cmd+1 through Cmd+0 — Switch tools
  • Cmd+O — Open file
  • Cmd+Z — Undo (in Annotate)
  • Cmd+Shift+Z — Redo (in Annotate)
  • Cmd+, — Preferences

Tips

  • Drag and drop PDF files directly onto the app window.
  • Password-protected files show a lock icon. Click to enter the password.
  • Output files are saved next to the original by default.
  • After an operation, click "Show in Finder" to locate your file.
